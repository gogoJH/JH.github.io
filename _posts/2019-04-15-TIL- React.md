### Props


==**Component**가 **Data**를 주고 받을 때 사용한다.==


 예를 들어,  `App`이라는 가장 상위 컴포넌트가 있다고 하면 하위 `Component`들에게 Data를 넘겨야 한다고 가정해보자.


```
const  App  = () => (
	<div>
		<h2>My Todo List</h2>
		<TodoList todos={[
			'Learn React',
			'Crush Recast.ly',
			'Maybe sleep'
			]}
		/>
	</div>
);

const  TodoList  = (props) => (
	<ul>
		<li>{props.todos[1]}</li>
		<li>{props.todos[2]}</li>
		<li>{props.todos[3]}</li>
	</ul>
);

ReactDOM.render(<App  />, document.getElementById('root'));
```
